Положите сюда приветственное изображение `welcome.jpg` (или укажите другой путь в `.env`).

Рекомендуемый размер: 1200×630 px или квадрат 1080×1080.
Форматы: JPG, PNG, WEBP.

В `.env`:
```
WELCOME_IMAGE_FILE=assets/welcome.jpg
```

Или по прямой ссылке:
```
WELCOME_IMAGE_URL=https://example.com/welcome.jpg
```
